# Practicum_projects

Содержит в себе четыре проекта:

1. **Помоги лично**  
2. **Продуктовый помощник**  
3. **Тестирование API**  
4. **Тестирование мобильного приложения Яндекс Метро**

## Помоги лично

Цель проекта — комплексное тестирование сайта благотворительного фонда «Помоги лично».

Задачи:
* проверить стабильность UI;
* убедиться в корректной работе API;
* выявить критические дефекты;
* оценить юзабилити основных сценариев.

В команде было 11 человек.

Мои задачи:
* тестирование API и UI;
* составление тест‑кейсов и баг‑репортов;
* поиск серых зон.

Используемые инструменты: *Postman*, *Swagger*.

Применяемые техники тест‑дизайна: классы эквивалентности (КЭ), граничные значения (ГЗ).

## Продуктовый помощник

Цель проекта — комплексное тестирование нового сервиса с рецептами, который позволяет разнообразить меню.

Команда состояла из 4 человек.

Мои задачи:
* организация команды;
* составление чек‑листа и баг‑репорта;
* поиск серых зон;
* составление общего отчёта.

Применяемые техники тест‑дизайна: классы эквивалентности (КЭ), граничные значения (ГЗ), mindmap.

## Тестирование API

Тестирование API приложения «Яндекс Прилавок».

Задачи:
* составление чек‑листа по методам: *POST*, *GET*, *PUT*, *DELETE*.

Используемые инструменты: *Postman*, *Swagger*.

## Тестирование мобильного приложения Яндекс Метро

Задачи:
* регрессионное тестирование Яндекс Метро через *Android Studio*;
* составление тест‑кейсов;
* составление чек‑листов.

Используемые инструменты: *Android Studio*.
